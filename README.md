✊ Rock Paper Scissors
A classic, interactive Rock-Paper-Scissors game built with a clean UI and real-time score tracking. This project focuses on DOM manipulation and conditional logic to provide a smooth user experience.

👉 Live Demo: https://zunaid990.github.io/rock-paper-scissors/

How It's Made:
Tech used: HTML, CSS, JavaScript

This application uses:

HTML for the game layout and display elements

CSS for the vibrant UI, hover effects, and responsive flexbox alignment

JavaScript for the core game logic, random computer selection, and dynamic score updates

Main Features:

Real-time Scoring: Tracks and displays wins for both the player and the computer.

Dynamic Feedback: Changes text color (Green for win, Red for loss) based on the round outcome.

Interactive UI: Large emoji buttons with smooth transition effects.

Instant Reset: Scores update instantly without needing to refresh the page.

Optimizations
Ternary Operators: Used clean ternary logic to determine win/loss conditions within the switch statement.

Class Toggle: Efficiently managed CSS classes (greenText, redText) using classList to keep the UI synchronized with game logic.

Flexbox Layout: Utilized Flexbox to ensure all game elements remain perfectly centered and readable.

Modular Logic: Separated the game choices into a constant array for easy maintenance.

Lessons Learned:
This project was a great way to practice Event Handling and DOM Manipulation. I learned how to effectively use switch statements to handle multiple game outcomes and how to manage global state (the scores) across multiple function calls. It also reinforced the importance of visual feedback in UX—using colors and large icons to make the game feel "alive."
